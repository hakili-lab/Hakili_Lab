## Exercice 1

Ecrire en notation scientifique le nombre suivant :

$$a = \frac{275 \times 10^{-4}}{0,025 \times 10^{-2}}$$

**Correction :**

$$a = \frac{275}{0,025} \times \frac{10^{-4}}{10^{-2}} = 11000 \times 10^{-2} = 1,1 \times 10^{4} \times 10^{-2} = 1,1 \times 10^{2}$$

**Réponse :** $$a = 1,1 \times 10^{2}$$

## Exercice 2

Comparer : $\sqrt{ab}$ et $\dfrac{a + b}{2}$ $(a > 0, b > 0)$

**Correction :**

$$\left(\frac{a + b}{2}\right)^2 - (\sqrt{ab})^2 = \frac{(a+b)^2}{4} - ab = \frac{a^2 - 2ab + b^2}{4} = \frac{(a-b)^2}{4} \ge 0$$

Donc $\left(\frac{a + b}{2}\right)^2 \ge (\sqrt{ab})^2$. Comme les deux nombres sont positifs :

$$\frac{a + b}{2} \ge \sqrt{ab}$$

**Réponse :** $$\frac{a + b}{2} \ge \sqrt{ab}$$ pour tout $a > 0$ et $b > 0$.

L'égalité a lieu si $a = b$.

## Exercice 3

Ecrire le plus simplement possible :

### A

$$A = 5\left[ ((( -2 - 1) + 3) - 6) + 5 \right] - 3\left( 2( -2 + 1) - 4 \right)$$

**Correction :**

$$A = 5\left[ (( -3 + 3) - 6) + 5 \right] - 3\left( 2(-1) - 4 \right)$$
$$A = 5\left[ (0 - 6) + 5 \right] - 3\left( -2 - 4 \right)$$
$$A = 5\left[ -6 + 5 \right] - 3\left( -6 \right)$$
$$A = 5(-1) + 18 = -5 + 18 = 13$$

**Réponse :** $$A = 13$$

### B

$$B = \sqrt{\frac{288}{169}} \times \frac{\sqrt{225}}{\sqrt{8}}$$

**Correction :**

$$B = \frac{\sqrt{288}}{\sqrt{169}} \times \frac{\sqrt{225}}{\sqrt{8}} = \frac{\sqrt{288} \times 15}{13 \times \sqrt{8}} = \frac{15}{13} \times \sqrt{\frac{288}{8}} = \frac{15}{13} \times \sqrt{36} = \frac{15}{13} \times 6 = \frac{90}{13}$$

**Réponse :** $$B = \frac{90}{13}$$

### C

$$C = \sqrt{(\sqrt{3} - 2)^2} - \sqrt{(\sqrt{3} + 2)^2} + 2\sqrt{3}$$

**Correction :**

On utilise $\sqrt{a^2} = |a|$.

$$|\sqrt{3} - 2| = 2 - \sqrt{3} \quad \text{car } \sqrt{3} < 2$$
$$|\sqrt{3} + 2| = \sqrt{3} + 2 \quad \text{car } \sqrt{3} + 2 > 0$$

$$C = (2 - \sqrt{3}) - (\sqrt{3} + 2) + 2\sqrt{3} = 0$$

**Réponse :** $$C = 0$$

### D

$$D = \sqrt{\sqrt{2} + 1} \times \sqrt{\sqrt{2} - 1}$$

**Correction :**

$$D = \sqrt{(\sqrt{2} + 1)(\sqrt{2} - 1)} = \sqrt{(\sqrt{2})^2 - 1^2} = \sqrt{2 - 1} = \sqrt{1} = 1$$

**Réponse :** $$D = 1$$

## Exercice 5

Calculer A et B en donnant les résultats sous forme de fraction irréductible.

### A

$$A = 1 + \frac{1}{2 + \frac{1}{\frac{3}{4} - \frac{1}{2}}}$$

**Correction :**

$$\frac{3}{4} - \frac{1}{2} = \frac{3}{4} - \frac{2}{4} = \frac{1}{4}$$
$$\frac{1}{\frac{1}{4}} = 4$$
$$2 + 4 = 6$$
$$A = 1 + \frac{1}{6} = \frac{7}{6}$$

**Réponse :** $$A = \frac{7}{6}$$

### B

$$B = \frac{9^{2} \times 4^{-3}}{2^{-5} \times 3^{3}}$$

**Correction :**

$$9^2 = (3^2)^2 = 3^4 \quad \text{et} \quad 4^{-3} = (2^2)^{-3} = 2^{-6}$$

$$B = \frac{3^{4} \times 2^{-6}}{2^{-5} \times 3^{3}} = 3^{4-3} \times 2^{-6 - (-5)} = 3^{1} \times 2^{-1} = \frac{3}{2}$$

**Réponse :** $$B = \frac{3}{2}$$

## Exercice 6

Ecrire sous forme de puissance entière de nombres premiers :

$$A = (27)^{-6} \times (3^{-5})^{-2} \times \left(\frac{1}{9}\right)^{-5}$$

**Correction :**

$$27 = 3^3 \Rightarrow (27)^{-6} = (3^3)^{-6} = 3^{-18}$$
$$(3^{-5})^{-2} = 3^{10}$$
$$\frac{1}{9} = 3^{-2} \Rightarrow \left(\frac{1}{9}\right)^{-5} = (3^{-2})^{-5} = 3^{10}$$

$$A = 3^{-18} \times 3^{10} \times 3^{10} = 3^{-18+10+10} = 3^{2}$$

**Réponse :** $$A = 3^{2}$$

## Exercice 9

Factoriser :

### A

$$A = (2x - 1)^{2} - (1 - 2x)$$

**Correction :**

$$A = (2x - 1)^{2} - (1 - 2x) = (2x - 1)^{2} + (2x - 1)$$
$$A = (2x - 1)[(2x - 1) + 1] = (2x - 1)(2x)$$

**Réponse :** $$A = 2x(2x - 1)$$

### B

$$B = (x - 1)^2 - (5 - 2x)^2$$

**Correction :**

$$B = [(x-1) - (5-2x)] \times [(x-1) + (5-2x)]$$
$$B = [x - 1 - 5 + 2x] \times [x - 1 + 5 - 2x]$$
$$B = [3x - 6] \times [4 - x] = 3(x - 2)(4 - x)$$

**Réponse :** $$B = 3(x-2)(4-x)$$

### C

$$C = x^2 + 7x + 12$$

**Correction :**

On cherche deux nombres dont la somme est 7 et le produit est 12 : 3 et 4.

$$C = x^2 + 3x + 4x + 12 = x(x+3) + 4(x+3) = (x+3)(x+4)$$

**Réponse :** $$C = (x+3)(x+4)$$

## Exercice 10

Résoudre dans $\mathbb{R}$ :

### a) $|x + 2| = |-2x + 6|$

$$|A| = |B| \iff A = B \text{ ou } A = -B$$

**Cas 1 :** $x + 2 = -2x + 6 \Rightarrow 3x = 4 \Rightarrow x = \frac{4}{3}$

**Cas 2 :** $x + 2 = 2x - 6 \Rightarrow -x = -8 \Rightarrow x = 8$

**Réponse :** $$S = \left\{ \frac{4}{3}, 8 \right\}$$

### b) $|-3x + 1| = -1$

Une valeur absolue est toujours $\ge 0$, donc ne peut pas être égale à $-1$.

**Réponse :** $$S = \emptyset$$

### c) $16 - x^2 = 4 - x$

$$(4 - x)(4 + x) = 4 - x$$
$$(4 - x)(4 + x) - (4 - x) = 0$$
$$(4 - x)(4 + x - 1) = 0$$
$$(4 - x)(3 + x) = 0$$

$4 - x = 0 \Rightarrow x = 4$ ou $3 + x = 0 \Rightarrow x = -3$

**Réponse :** $$S = \{-3, 4\}$$

### d) $-x^2 + 2x + 3 < 0$

$$-x^2 + 2x + 3 < 0 \iff x^2 - 2x - 3 > 0$$
$$x^2 - 2x - 3 = (x+1)(x-3)$$

Tableau de signes :

| $x$ | $-\infty$ | $-1$ | $3$ | $+\infty$ |
|:---|:---:|:---:|:---:|:---:|
| $x+1$ | $-$ | $0$ | $+$ | $+$ |
| $x-3$ | $-$ | $-$ | $0$ | $+$ |
| Produit | $+$ | $0$ | $-$ | $0$ | $+$ |

**Réponse :** $$S = ]-\infty, -1[ \cup ]3, +\infty[$$

### e) $\left| x - \frac{1}{2} \right| \le \frac{13}{2}$

$$-\frac{13}{2} \le x - \frac{1}{2} \le \frac{13}{2}$$
$$-\frac{13}{2} + \frac{1}{2} \le x \le \frac{13}{2} + \frac{1}{2}$$
$$-6 \le x \le 7$$

**Réponse :** $$S = [-6, 7]$$

### f) $\sqrt{2} \cos x = 1$

$$\cos x = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$$

**Réponse :** $$S = \left\{ \frac{\pi}{4} + 2k\pi, -\frac{\pi}{4} + 2k\pi \mid k \in \mathbb{Z} \right\}$$

## Exercice 12

On donne : $\sin x = \frac{4}{5}$. Calculer $\cos x$ et $\tan x$ pour $x \in ]0; \frac{\pi}{2}[$

**Correction :**

$$\cos^2 x = 1 - \sin^2 x = 1 - \left(\frac{4}{5}\right)^2 = 1 - \frac{16}{25} = \frac{9}{25}$$

Comme $x \in ]0; \frac{\pi}{2}[$, $\cos x > 0$, donc $\cos x = \frac{3}{5}$.

$$\tan x = \frac{\sin x}{\cos x} = \frac{\frac{4}{5}}{\frac{3}{5}} = \frac{4}{3}$$

**Réponse :** $$\cos x = \frac{3}{5} \quad \text{et} \quad \tan x = \frac{4}{3}$$

On donne : $\|\overrightarrow{u}\| = 2$, $\|\overrightarrow{v}\| = \sqrt{3}$ et $\text{Mes}(\widehat{\overrightarrow{u},\overrightarrow{v}}) = \frac{\pi}{6}$.

## Exercice 16

Résoudre dans $\mathbb{R} \times \mathbb{R}$ :

$$\begin{cases} \frac{x}{3} + \frac{y}{5} = 1 \\ \frac{x}{2} - \frac{y}{2} = 1 \end{cases}$$

**Correction :**

**Équation 1 :** $\frac{x}{3} + \frac{y}{5} = 1 \quad \underset{\times 15}{\Longrightarrow} \quad 5x + 3y = 15$

**Équation 2 :** $\frac{x}{2} - \frac{y}{2} = 1 \quad \underset{\times 2}{\Longrightarrow} \quad x - y = 2$

On résout : $\begin{cases} 5x + 3y = 15 \\ x - y = 2 \end{cases}$

De la deuxième équation : $x = y + 2$.

On remplace dans la première :

$$5(y + 2) + 3y = 15$$
$$5y + 10 + 3y = 15$$
$$8y = 5$$
$$y = \frac{5}{8}$$

On en déduit $x$ :

$$x = y + 2 = \frac{5}{8} + 2 = \frac{5}{8} + \frac{16}{8} = \frac{21}{8}$$

**Réponse :** $$S = \left\{ \left( \frac{21}{8}, \frac{5}{8} \right) \right\}$$
